# 数字孪生校园交互系统 — 浙江树人学院拱宸桥校区

> 浙江树人学院《计算机图形学》期中考核作业

## 📋 项目简介

本项目是一个基于 **Three.js** 的数字孪生校园交互系统，使用 HTML + JavaScript 构建，可在浏览器中直接运行。系统以**浙江树人学院拱宸桥校区**（杭州市拱墅区树人街8号）为原型，按照真实校园地图和建筑布局，构建了包含南校区、北校区、舟山东路在内的完整 3D 校园场景。支持鼠标交互浏览、点击查看建筑详细信息、多种视角切换、图层控制以及喷泉粒子动画等丰富的交互功能。

## ✨ 功能介绍

### 🎮 交互控制（≥3类）
| 交互方式 | 说明 |
|---------|------|
| 鼠标拖拽旋转 | 按住左键拖动旋转视角 |
| 鼠标滚轮缩放 | 滚动滚轮拉近/拉远视角 |
| 右键平移 | 按住右键拖动平移场景 |
| 点击建筑信息 | 点击任意建筑弹出详细信息面板 |

### 📷 视角切换（4种）
| 视角 | 说明 |
|------|------|
| 🔍 透视视角 | 默认自由视角，从东南方向俯瞰校园 |
| 🦅 鸟瞰视角 | 正上方俯视，查看南北校区全貌 |
| 🏛️ 南门视角 | 面向南校门，展示入口区域 |
| 🏟️ 北校区视角 | 面向北校区，展示体育馆和操场 |

### 🔧 核心功能（≥3个）
1. **图文介绍面板** — 点击建筑弹出信息卡片，展示真实建筑名称、类型和详细描述
2. **图层切换控制** — 独立控制建筑/道路/植被/设施四类图层的显示与隐藏
3. **视角变换系统** — 一键切换透视/鸟瞰/南门/北校区四种预设视角（平滑过渡动画）

### 🎬 动画效果
- 南门喷泉粒子系统（水花循环动画）
- 旗帜飘动效果
- 视角切换平滑过渡动画

## 🛠️ 技术架构

| 层级 | 技术/工具 |
|------|----------|
| 前端框架 | HTML5 + CSS3 + JavaScript (ES Module) |
| 3D引擎 | Three.js v0.179.0 |
| 相机控制 | OrbitControls（阻尼旋转、缩放、平移） |
| 光照系统 | 方向光（阴影映射2048x2048）+ 环境光 + 半球光 |
| 渲染特性 | PCFSoftShadowMap + ACES Filmic 色调映射 |
| AI工具 | Trae IDE (AI辅助编程) |

## 🚀 运行方式

由于使用了 ES Module，需要通过 HTTP 服务运行：

```bash
# 使用 Node.js
npx serve . -p 3000

# 或使用 VS Code Live Server 插件打开 index.html
```

浏览器打开 `http://localhost:3000`。

## 📝 操作指南

- **旋转视角**：鼠标左键拖拽
- **缩放场景**：鼠标滚轮
- **平移场景**：鼠标右键拖拽
- **查看信息**：点击任意建筑
- **切换视角**：点击顶部工具栏按钮
- **图层控制**：点击左侧面板开关
